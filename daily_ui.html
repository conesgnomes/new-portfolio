Daily UI Challenge

UI Design

Overview

In an effort to up level my user interface design skills and increase the number of UI projects in my portfolio, I signed up for the Daily UI Challenge, a 100-day challenge sponsored by dailyui.co. Every morning I wake up with a new UI design problem to solve in my inbox in whatever way I see fit. The prompts are purposefully brief and open-ended, allowing for maximum creativity, which makes them all the more interesting and fun to complete. I have chosen to complete each in mobile size because I enjoy designing for mobile most — I like the extra challenge the minimized screen real estate provides and I like to think doing so stretches my abilities and skills. I have chosen a selection of my favorite designs to share below. I have included the prompt with each.
